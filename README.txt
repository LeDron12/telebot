Бот является альтернативой ленты в ВК. Бот подхватывает
посты из открытых пабликов и по команде refresh
выплевывает их пользователю. Бот решает проблему того,
что некоторые пользователи вк хотят полностью перейти на телегу,
но не переходят, т.к. мемы смотреть хочется в общей ленте))
Использовал Telegram API, VK API
Для работы нужен JSON файл vk_data с вашим логином и паролем и api_key.txt c google ключом.
